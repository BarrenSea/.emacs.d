** Setup
#+begin_src shell
$  git clone https://github.com/donjuanplatinum/.emacs.d ~/.emacs.d
$  emacs -q init.org
$  在emacs中输入 M-x org-babel-tangle
  bash install.sh
  #安装librime与typst-preview
$  cargo install typst-preview
$  cargo install typst-cli
$  emerge librime #gentoo
#+end_src
